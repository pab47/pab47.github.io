  • Getting started with latex
    • Windows 1) Download miketex (https://miktex.org/) and install; 2) Download texStudio and install (http://www.texstudio.org/)
    • Mac 1) Download and install miketex for mac (https://miktex.org/howto/install-miktex-mac); 2) Download and install texShop (http://pages.uoregon.edu/koch/texshop/)
    • After installing mikeTex and editor (texStudio or texShop), download this latex example file and open the template.tex file in the editor and press the green button that says ">" on texStudio or type Apple key + T on texShop. This will create a pdf file template.pdf, which is the final result.
